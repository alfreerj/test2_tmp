Связываем папку с удаленным репозиторием
git remote add origin https://github.com/LeoXCoder/test_project.git

Закачиваем код на удаленный репозиторий
git push

добавить удалённый репозиторий и присвоить ему имя (shortname)
git remote add [shortname] [url]
$ git remote add pb https://github.com/paulboone/ticgit
Новый репозиторий надо создать на github, через интерфейс в браузере
потом выйдет подсказка с вариантами комманд: 
	1. - создать новый локальный (на компе) репозторий и
	   - связать его с удаленным (на сервере) репозиторием
	или
	2. - связать уже существующий локальный (на компе) репозторий с удаленным (на сервере) репозиторием
 
Удалить репозиторий на сервере:
	- выбираем и заходим в репозиторий
	- вкладка Settings
	- раздел (внизу) Danger Zone
	- удалить Delete this repository
	- вводим имя репозиторий для подтверждения и удаляем


Убедить git, что файл конфига не меняется:
git update-index --assume-unchanged super.conf
